## ✨ Форматування Markdown

### 🔹 Жирний, курсив, закреслений
**Жирний текст**  
*Курсивний текст*  
~~Закреслений текст~~

### 🔹 Індекси
Хімічна формула: H<sub>2</sub>O  
Математична формула: X<sup>2</sup>

### 🔹 Підкреслення
<ins>Це підкреслений текст</ins>
